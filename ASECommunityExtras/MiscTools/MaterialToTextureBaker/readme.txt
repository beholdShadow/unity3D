Editor window created via tutorial at http://www.ronja-tutorials.com/2018/10/13/baking_shaders.html
twitter: https://twitter.com/totallyRonja 

What: 
Editor window that lets you assign a material and bake it out to a texture. Very useful for exporting noise textures created using ase, or even combing texture masks into RGBA channels, etc! To create a material to use as the "baker", it needs to be an unlit shader.
